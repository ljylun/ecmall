一、填空题

第1题

A = [ [ 1, 0, i], [ 0, 1, 2], [-i, 2, 5] ] x = [1,1,i]
最终答案： Ax = [0,1+2i,2+4i]^T ‖Ax‖₂ = 5 ‖A‖₁ = 8 ‖A‖∞ = 8

详细解答

由于矩阵乘法要求 x 为列向量，因此实际计算时取

x = [1 1 i]

计算 Ax：

Ax = [ 1·1 + 0·1 + i·i, 0·1 + 1·1 + 2·i, (-i)·1 + 2·1 + 5·i ]

逐项计算：

第一项： 1 + i² = 1 - 1 = 0 第二项： 1 + 2i 第三项： -i + 2 + 5i = 2 + 4i
因此：
Ax = [0, 1+2i, 2+4i]^T

计算二范数：

‖Ax‖₂ = √( |0|² + |1+2i|² + |2+4i|² ) = √(0 + 5 + 20) = √25 = 5

计算矩阵 1-范数：

第1列： |1| + |0| + |-i| = 2 第2列： |0| + |1| + |2| = 3 第3列： |i| + |2| + |5| = 8
因此：
‖A‖₁ = 8

计算矩阵 ∞-范数：

第1行： |1| + |0| + |i| = 2 第2行： |0| + |1| + |2| = 3 第3行： |-i| + |2| + |5| = 8
因此：
‖A‖∞ = 8
考察知识点：
  • 复矩阵运算
  • 复向量二范数
  • 矩阵1范数
  • 矩阵∞范数
  • Hermite矩阵
易错点：
  • 忽略 i² = -1
  • 复数模平方计算错误
  • 1范数与∞范数混淆

第2题

A = (1/10) [ [1,2], [8,1] ]
答案： {A^k} 收敛，并且 A^k → O

详细解答

设

B = [ [1,2], [8,1] ]

则 A = (1/10)B。

求 B 的特征值：

det(λI-B) = (λ-1)² -16 = λ² -2λ -15 = (λ-5)(λ+3)
因此 B 的特征值为：
λ1 = 5 λ2 = -3
于是 A 的特征值为：
1/2 与 -3/10
谱半径：
ρ(A)=1/2<1
因此：
A^k → O
考察知识点：
  • 矩阵幂收敛
  • 谱半径
  • 特征值判定

第3题

A = [ [1,2,0], [2,1,3], [0,2,1] ]
答案： A 不可约

详细解答

构造有向图：

  • 1 ↔ 2
  • 2 ↔ 3

因此：

  • 1 可达 3
  • 3 可达 1

图强连通，因此矩阵不可约。

考察知识点：
  • 不可约矩阵
  • 有向图强连通

二、计算题

第1题

已知向量组的度量矩阵：

Gα= [ [1,-1,0], [-1,2,-1], [0,-1,1] ]
最终答案： Ge= [ [1/4,-1/4,-1/2], [-1/4,5/4,1/2], [-1/2,1/2,1] ] [u,v]=9/4

详细解答

构造过渡矩阵：

P= [ [1,1,0], [1,-1,0], [0,0,1] ]

计算逆矩阵：

P⁻¹= [ [1/2,1/2,0], [1/2,-1/2,0], [0,0,1] ]

利用度量矩阵变换公式：

Ge=P^{-T}GαP^{-1}
得到：
Ge= [ [1/4,-1/4,-1/2], [-1/4,5/4,1/2], [-1/2,1/2,1] ]
随后：
[u,v]=u^TGev
最终得到：
[u,v]=9/4

三、证明题

第1题

证明： 利用 Cayley-Hamilton 定理： A³-A²-A+E=0 即： A³=A²+A-E

然后利用数学归纳法即可证明：

A^n=A^{n-2}+A²-E

1≤‖A‖·‖A⁻¹‖
